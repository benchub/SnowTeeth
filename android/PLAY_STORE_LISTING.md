## Store Listing Assets Required

### 1. App Icon (Required)
- **Size:** 512 x 512 pixels
- **Format:** 32-bit PNG (with alpha)
- **Max file size:** 1024 KB
- **Note:** Use your existing ic_launcher but export at 512x512

### 2. Feature Graphic (Required)
- **Size:** 1024 x 500 pixels
- **Format:** JPEG or 24-bit PNG (no alpha)
- **Max file size:** 1024 KB
- **Design suggestions:**
  - Show the flame visualization or yeti character
  - Include app name
  - Tagline: "Track Your Adventures. Keep Your Privacy."
  - Use snow/mountain imagery

### 3. Phone Screenshots (Required)
- **Minimum:** 2 screenshots
- **Maximum:** 8 screenshots
- **Dimensions:** Any of these:
  - 1080 x 1920 (16:9 portrait)
  - 1080 x 2340 (longer displays)
  - 1440 x 2960 (high-res)

**Recommended screenshot order:**
1. **Home Screen** - Show main navigation with 4 buttons
2. **Flame Visualization** - Show the fire effect in action
3. **Data Visualization** - Show GPS data display with speed
4. **Yeti Visualization** - Show animated yeti character
5. **Stats Screen** - Show statistics with actual data
6. **Configuration Screen** - Show threshold settings
7. **Tracking Notification** - Show active tracking
8. **Privacy Message** - Highlight "All data stays on your device"

**Screenshot design tips:**
- Add captions/overlays explaining each feature
- Use actual app screenshots (not mockups)
- Show app in use with realistic data
- Consider adding text overlays: "Real-time Speed", "Track Your Runs", etc.

### 4. Tablet Screenshots (Optional but recommended)
- Similar to phone but tablet-optimized dimensions
- Minimum 7-inch tablet: 1024 x 1920
- 10-inch tablet: 2048 x 2732

### 5. Promo Video (Optional)
- **Length:** 30 seconds to 2 minutes
- **Aspect ratio:** 16:9 or 9:16
- **Format:** YouTube URL
- **Suggested content:**
  - Quick tour of all 4 screens
  - Show visualization modes
  - Demonstrate stats
  - End with privacy message
